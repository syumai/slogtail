# lduck

Hono RPC + React CLI tool. Starts a built-in web server with a React frontend.

## Setup

```bash
pnpm install
```

## Scripts

```bash
pnpm dev          # Start dev server
pnpm build        # Build for production
pnpm test         # Run tests
pnpm lint         # Lint + type check
pnpm format       # Format code
```

## Production

```bash
pnpm build
node dist/index.js
```
